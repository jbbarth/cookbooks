h1. Description

This cookbook provides recipes to install and configure "RVM (Ruby Version Manager)":http://rvm.beginrescueend.com/. RVM is installed at system level ("system-wide install").

h1. Definitions

h2. rvm_ruby

Manage a ruby environment by running @rvm <action> <version>@ behind the scenes.

h3. Actions

|_. Action|_. Description|_. Default|
|*install*|Install a ruby environment|Yes|
|*remove*|Remove a ruby environment, keeping sources||
|*purge*|Remove a ruby environment including sources||

h3. Attributes

|_.Attribute|_.Description|_.Default value|
|*version*|_Name attribute_: Ruby implementation and version to install/uninstall|name|

h3. Examples

<pre>
rvm_ruby "1.8.7"

rvm_ruby "uninstall 1.8.6" do
  version "ruby-1.8.6"
  action  :purge
end
</pre>

h2. rvm_gemset

Manage ruby gemsets inside a ruby environment (possibly created with @rvm_ruby@ resource).

h3. Actions

|_. Action|_. Description|_. Default|
|*create*|Create a gemset|Yes|
|*delete*|Delete a gemset||

h3. Attributes

|_.Attribute|_.Description|_.Default value|
|*gemset_name*|_Name attribute_: Name of the gemset|name|

h3. Examples

<pre>
#creates a gemset named 'shiny_gemset' in the ruby 1.9.2 environment
rvm_gemset "1.9.2@shiny_gemset"

#then delete it
rvm_gemset "1.9.2@shiny_gemset" do
  action :delete
end
</pre>
